# LED Dimmer (или как я пытался управлять светодиодами)

Небольшая программа на Python с GUI на Tkinter.  
Задумывалась как контроллер LED через Arduino,  
по факту — пока что красивое окошко с большими амбициями.

---

## Что это вообще такое

Это попытка сделать:
- ввод времени работы светодиода
- регулировку яркости
- кнопки для синего и красного LED
- и всё это через Arduino

На текущий момент:
- Arduino **не подключён**
- `pyFirmata` **закомментирован**
- логика LED **ждёт своего часа**
- GUI **работает и не падает** (уже победа)

---

## Что реально работает

✔ Окно Tkinter  
✔ Поле ввода времени  
✔ Ползунок яркости  
✔ Кнопки (они есть, нажимаются, выглядят уверенно)  
✔ Кнопка «Справка», которая честно говорит правду

✖ Управление LED
