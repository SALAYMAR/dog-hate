# LED Dimmer (свет в будке)

![Свет в будке](https://i.pinimg.com/736x/e5/e9/47/e5e9479cfa2e489c9ef363f0c6539510.jpg)

Небольшая программа на Python с GUI на Tkinter.  
Задумывалась как контроллер LED через Arduino,  
по факту — пока что красивейшее окошко с большими амбициями.

---

## Что это 

Это окно где есть:
- ввод времени работы светодиода
- регулировку яркости
- кнопки для синего и красного LED
- и всё это через Arduino

На текущий момент:
- Arduino **не подключён**
- `pyFirmata` **закомментирован**
- логика LED **ждёт своего часа**
- GUI **работает и не падает** (уже победа)

## Окно со справкой

```bash
def aboutMsg():
    messagebox.showinfo(
        "Это программное обеспечение, которому всё равно на логику\nLED Контроллер Вер 1.0\nJanuary 2026"
    )
```
Показывает всплывающее окно со справкой.
Честно говорит пользователю, что программа немного криво, но живёт.
На данный момент — единственное место, где программа не врет.

## Создание главного окна
```bash
win = tk.Tk()
win.title("Dimmer LED")
win.minsize(235, 150)
```
tk.Tk() — создаёт главное окно программы

title — название окна

minsize — минимальный размер окна, чтобы кнопки и ползунок не растекались

## Поле ввода времени работы LED
```bash
LEDtime = tk.Entry(win, bd=6, width=8)
LEDtime.grid(column=1, row=1)
tk.Label(win, text="LED ВКЛ Время(сек)").grid(column=2, row=1)
```
Entry — текстовое поле для ввода времени в секундах

grid — размещение элемента на сетке окна

Label — подпись для поля, чтобы пользователь понимал, что куда вводить

## Ползунок яркости 
```bash
LEDbright = tk.Scale(win, bd=5, from_=0, to=100, orient=tk.HORIZONTAL)
LEDbright.grid(column=1, row=2)
tk.Label(win, text="Яркость LED")
```
Scale — горизонтальный ползунок от 0 до 100

Показывает выбранную яркость

На данный момент только визуально, реально на LED не влияет

## Кнопки управления LED
```bash
Bluebtn = tk.Button(win, bd=4, text="Blue LED")  # пока просто кнопка
Bluebtn.grid(column=1, row=3)

redbtn = tk.Button(win, bd=4, text="RED LED")   # пока просто кнопка
redbtn.grid(column=2, row=3)
```
Blue и Red кнопки — планировались для включения LED через Arduino

Сейчас только внешний вид, физического эффекта нет

С эстетикой всё ок, кнопки уверенно сидят на своих местах

## Кнопки справки и выхода
```bash
aboutBtn = tk.Button(win, text="Справка", command=aboutMsg)
aboutBtn.grid(column=1, row=4)

quitBtn = tk.Button(win, text="Выйти")  # пока не подключено win.quit
quitBtn.grid(column=2, row=4)
```
Cправка — вызывает aboutMsg, честно рассказывает про программу

Выйти — пока просто кнопка, без действия, но место заняла
